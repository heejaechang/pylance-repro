# pylance-repro

Auto-generated repository of reproduction workspaces for
[microsoft/pylance-release](https://github.com/microsoft/pylance-release) issues.

Each verified issue is published under pylance-issues/<issue-number>/ by the
Pylance issue-intake monitor. Workspaces are reconstructable from their
equirements.txt / pyproject.toml; installed environments (.venv,
site-packages) and build artifacts are intentionally **not** committed.
